# Sió Viu – Web

Pàgina web de l'Associació per la Defensa Ambiental Territorial
de Tarroja de Segarra, la Prenyanosa i pobles veïns.

## Estructura del projecte

```
SioViu/
├── index.html          ← Pàgina principal
├── css/
│   └── style.css       ← Tots els estils
├── js/
│   └── main.js         ← Interactivitat (burger, scroll actiu)
├── images/
│   ├── logo-sioviu.png
│   ├── hero-bg.jpg
│   ├── territori.jpg
│   ├── manifest.jpg
│   ├── planta-biogas.jpg
│   ├── mapa-zona.jpg
│   ├── noticia-segre.png
│   └── noticia-ara.png
└── .vscode/
    ├── settings.json   ← Configuració VS Code
    └── extensions.json ← Extensions recomanades
```

## Com obrir en local

1. Obre la carpeta `SioViu` amb VS Code
2. Instal·la l'extensió **Live Server** (recomanada automàticament)
3. Clic dret a `index.html` → **Open with Live Server**
4. S'obrirà al navegador a `http://127.0.0.1:5500`

## Com publicar (Netlify)

1. Ves a [netlify.com/drop](https://app.netlify.com/drop)
2. Arrossega tota la carpeta `SioViu`
3. En 30 segons tens URL pública
